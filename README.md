# Homebrew Tap for Clawbreaker

Official Homebrew tap for [Clawbreaker](https://clawbreaker.ai) CLI.

## Installation

```bash
brew tap clawbreaker/clawbreaker
brew install claw
```

Or install directly:

```bash
brew install clawbreaker/clawbreaker/claw
```

## Usage

```bash
# Connect to your Clawbreaker instance
claw login

# Start a chat session
claw chat

# List sessions
claw sessions list
```

## Updating

```bash
brew upgrade claw
```

## Uninstalling

```bash
brew uninstall claw
brew untap clawbreaker/clawbreaker
```
